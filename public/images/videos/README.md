## 🚀 Firebase申請での使用

### 推奨用途
- **Firebase App Distribution:** MP4形式使用（ファイルサイズ小）
- **Google Play Store:** MP4形式推奨
- **ウェブサイト/ブログ:** MP4形式（帯域幅効率）
- **SNS共有:** MP4形式（互換性向上）
- **GIF仕様が必須の場合:** GIF形式使用

### 品質パラメータ
```
MP4変換設定:
- Codec: H.264 (libx264)
- Preset: slow (高品質)
- CRF: 23 (品質/サイズのバランス)
- Pixel Format: yuv420p (標準互換性)
- Movflags: +faststart (プログレッシブダウンロード対応)
```

## 📝 ナレーション・BGM追加（将来対応）

ユーザーが別途AI音声生成ツール（Google TTSなど）で準備したナレーション・BGMを使用して、YouTube/プロモーションビデオ版（統合版）を作成可能：

```bash
# 必要なファイル（ユーザー提供）
- /path/to/narration.mp3 （AI音声生成）
- /path/to/bgm.mp3 （ロイヤリティフリー音源）

# 編集コマンド例（ffmpeg）
ffmpeg -i combined_with_text.mp4 \
  -i narration.mp3 -i bgm.mp3 \
  -filter_complex "[1:a]volume=1.0[narration]; [2:a]volume=0.3[bgm]; [narration][bgm]amix=inputs=2[audio]" \
  -map 0:v -map "[audio]" \
  -c:v copy -c:a aac \
  taskel_pv_final.mp4
```
