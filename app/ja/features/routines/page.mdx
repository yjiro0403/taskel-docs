
# ルーチン (Routines)

繰り返しのタスクを自動化し、習慣化をサポートします。

## ルーチンの作成

1. サイドバーから **"Routines"** メニューを選択します。
2. `New Routine` ボタンをクリックします。
3. **頻度（Frequency）** を設定します。
    - **Daily**: 毎日（曜日指定も可能）
    - **Weekly**: 毎週特定の日
    - **Monthly**: 毎月特定の日
4. 時間やプロジェクトなどの詳細を設定して保存します。

## 仮想タスクと実体化

- ルーチンを設定すると、カレンダー上に **「仮想タスク（Virtual Task）」** として淡く表示されます。
- その日が来たり、編集したりすると **「実体タスク（Real Task）」** に変わり、通常のタスクと同じように操作できるようになります。
