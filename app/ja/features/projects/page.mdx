
# プロジェクト管理 (Projects)

タスクをプロジェクト単位で管理し、進捗を可視化します。

## プロジェクトの作成

1. サイドバーの **"Projects"** セクションにある `+` ボタンをクリックします。
2. プロジェクト名を入力します。
    - 色やアイコンを設定して視認性を高めることもできます。

## メンバーの招待

Taskelでは、プロジェクトごとにメンバーを招待して共同作業ができます（プロプラン等）。

1. プロジェクト設定画面を開きます。
2. **"Members"** タブを選択します。
3. 招待したい人のメールアドレスを入力して `Invite` をクリックします。
4. または、**"Copy Link"** で招待リンクを生成し、チャットツールなどで共有します。
