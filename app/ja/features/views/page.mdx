
# ビューの切替 (Views)

Taskelには、目的に合わせた多様なビューが用意されています。

## Daily View (推奨)
- **ショートカット**: `D`
- **用途**: 今日のタスクを実行するためのメイン画面。
- **特徴**: 時間の流れを意識しながら、目の前のタスクに集中できます。

## Weekly View
- **ショートカット**: `W`
- **用途**: 今週の予定を確認・調整する。
- **特徴**: 1週間のタスクの負荷状況を俯瞰できます。

## Monthly View
- **ショートカット**: `M`
- **用途**: 月単位の大きなイベントや締め切りを確認する。
- **特徴**: カレンダー形式で全体のスケジュールを把握できます。

## Yearly View
- **ショートカット**: `Y`
- **用途**: 年間の目標や大きなマイルストーンを設定する。
- **特徴**: 1年を通じた長期的な視点を持つことができます。

## Planning View
- **ショートカット**: `P`
- **用途**: 未整理のタスク（Inbox）を整理し、日付を割り振る。
- **特徴**: 右側にInbox、左側にカレンダーが表示され、ドラッグ&ドロップで計画を立てられます。
