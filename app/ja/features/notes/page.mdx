
# ノート / 振り返り (Notes & Review)

日々の気づきや振り返りを記録するための機能です。

## 日次ノート (Daily Notes)

- Daily Viewの日付バーにある **ドキュメントアイコン** をクリックすると、その日のノートが開きます。
- Markdown形式で自由に記述できます。
- 自動保存されるため、保存操作は不要です。

## レポートの作成

- ノート画面の **"Copy Report"** ボタンをクリックすると、その日の「完了タスク」と「ノートの内容」をまとめたテキストがクリップボードにコピーされます。
- 日報や週報としてSlackなどに貼り付けるのに便利です。

## 周期的な振り返り

- **Weekly View**: 週の振り返りノートを作成できます。今週の成果やハイライトを記録。
- **Monthly View**: 月単位の振り返りノート。テーマや目標の進捗を振り返り。
- **Yearly View**: 年間の抱負や大きな振り返りを記録。
