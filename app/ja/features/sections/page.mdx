
# セクション管理

1日を「Morning」「Afternoon」「Evening」などのセクションに区切ることで、リズムを作ります。

## セクションの追加・編集

1. 日付バーの右側にある歯車アイコン（設定）をクリックします。
2. **"Add New Section"** ボタンで新しいセクションを追加できます。
3. セクション名（例: "Focus Time"）や開始時間（例: "10:00"）を編集します。
4. `Save` すると、メインのタスクリストに反映されます。

## スケジュールへの反映

セクションに設定した開始時間は、タイムラインビューでのタスク配置の目安になります。
