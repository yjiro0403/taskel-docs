
# キーボードショートカット

## ナビゲーション

- `D`: Daily View
- `W`: Weekly View
- `M`: Monthly View
- `Y`: Yearly View
- `P`: Planning View
- `A`: Analytics (未実装の場合あり)

## タスク操作

- `N`: 新しいタスクを作成
- `Cmd/Ctrl + Enter`: タスク作成・編集の保存
- `Esc`: モーダルを閉じる

## データ操作

- `Cmd/Ctrl + Z`: Undo (取り消し)
- `Cmd/Ctrl + S`: 手動保存 (通常は自動保存)
