
# おすすめワークフロー

Taskelを使った理想的な1日の流れ（例）を紹介します。

## 朝: 計画 (Planning)

1. **Daily View** を開きます。
2. 前日の未完了タスクや、今日やるべきタスクを確認します。
3. **Morning / Afternoon / Evening** の各セクションにタスクを振り分けます。
4. 「今日絶対に終わらせるタスク」を決めたら、準備完了です。

**AI活用**: AIチャットで「今日のタスクを見せて」と聞くと、現状を素早く把握できます。「今週の目標を教えて」で優先度の確認も。

## 日中: 実行 (Execution)

1. 最初のタスクの **Playボタン（▶）** を押します。
2. 集中して作業します。
3. 終わったら **停止ボタン（■）** を押して完了にし、次のタスクをスタートします。
4. 思いついたアイデアやタスクは、`N` キーですぐにタスクを追加し、今の作業に戻ります。

**AI活用**: 「今からメール返信を始める」とAIに伝えると、タスク作成と同時にタイマーが即スタート。割り込み作業が入っても「割り込みで電話対応」でスムーズに切り替えられます。

## 夜: 振り返り (Review)

1. 全てのタスクが完了したか確認します。
2. **Daily Note** アイコン（日付バーのドキュメントアイコン）をクリックし、今日の気づきをメモします。
3. 明日の予定を軽く確認して終了します。

**AI活用**: 「今日の振り返りをして」とAIに聞くと、完了率・見積もり精度・目標進捗をまとめたレポートが自動生成されます。「見積もりの精度を確認したい」で時間感覚のチューニングも。

---

## 週末: 週次レビュー

1. **Weekly View** で今週の成果を俯瞰します。
2. 未完了タスクを来週に持ち越すか判断します。
3. 来週の目標とタスクを計画します。

**AI活用**: 「今月の目標を分解して」で、目標から具体的なタスク候補を一括提案してもらえます。
