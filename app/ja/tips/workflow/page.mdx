
# おすすめワークフロー

Taskelを使った理想的な1日の流れ（例）を紹介します。

## 朝: 計画 (Planning)

1. **Daily View** を開きます。
2. 前日の未完了タスクや、今日やるべきタスクを確認します。
3. **Morning / Afternoon / Evening** の各セクションにタスクを振り分けます。
4. 「今日絶対に終わらせるタスク」を決めたら、準備完了です。

## 日中: 実行 (Execution)

1. 最初のタスクの **Playボタン** を押します。
2. 集中して作業します。
3. 終わったらチェックを入れて完了にし、次のタスクをスタートします。
4. 思いついたアイデアやタスクは、すぐにショートカットでInboxに追加し、今の作業に戻ります。

## 夜: 振り返り (Review)

1. 全てのタスクが完了したか確認します。
2. **Daily Note** アイコンをクリックし、今日の気づきをメモします。
3. 明日の予定を軽く確認して終了します。
