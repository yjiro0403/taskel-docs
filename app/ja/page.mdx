
# Taskel ドキュメント

> **目標と今日が、つながり続ける。**

**あなたの1年を、今日の30分に変える。**

Taskelは「タスク管理ツール」ではありません。
**年間目標と今日のアクションを双方向で接続し、実行・計測・振り返りの循環を毎日回すことで、目標と自分自身が磨かれていく体験**を提供するアプリです。

![Daily View](/images/daily-view.png)

---

## Taskelの7つの柱

| 柱 | 体験 |
|---|---|
| 🎯 **目標の双方向接続** | 年→月→週→今日が一本の線。実行が目標を磨く |
| ⏰ **時間の現実の可視化** | 終了予定時刻で「入りきらない」が5秒でわかる |
| ▶️ **実行の駆動** | ワンクリックタイマーで計画から実行への橋を架ける |
| 🪞 **自己校正（Time Mirror）** | 見積もり vs 実績で時間感覚を磨く。AIは鏡 |
| 🛡️ **心理的安全** | 未完了 = 罪にしない。Chuteで翌日に流せる |
| 🔄 **習慣の自動化** | ルーチンが自動で並ぶ。決断疲れの排除 |
| 📊 **行動選択の最適化** | スコアリングで「次に何をやるか」を根拠で選ぶ |

→ [7つの柱の詳細を見る](/ja/concept/what-is-taskel)

## Taskelメソッド — 毎日回す8ステップ

1. **年間目標を立てる**
2. **月→週→今日に落とす**
3. **今日のタスクを時間幅で並べる**
4. **見積もり時間をつける**
5. **タイマーで実行する**
6. **終わらなければChute（先送り）**
7. **Time Mirrorで振り返る**
8. **今日の気づきで目標を見直す**

→ [Taskelメソッドの詳細を見る](/ja/concept/taskel-method)

## こんな方におすすめ

- 仕事ではOKRやKPIで目標を回しているのに、**自分の人生にはTodoリストしか使っていない**
- フリーランスや経営者で、**目標はあるが日々のタスクに追われて年初の目標が消えている**
- **Todoistを3回入れ直した**経験がある
- **「今日やること」と「本当にやりたいこと」が断絶**している
- 見積もりが苦手で、**いつも「時間が足りない」**と感じている
- タスク管理ツールの**Overdueの赤字に疲れた**

## 設計哲学

### Assist, not Delegate（支援するが、代行しない）
AIは「鏡」。見積もりを代行せず、優先順位を勝手に変えない。自分で決めるから、磨かれる。

### 未完了は罰ではない
Chute（先送り）は正式な機能。「Overdue」の赤字地獄から解放されます。

### 目標は生き物である
変わったから失敗なのではなく、変わったから成長している。目標の変化を歓迎します。
