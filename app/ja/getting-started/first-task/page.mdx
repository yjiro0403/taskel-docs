
# 最初のタスクを作る

タスクの作成は非常にシンプルです。

## クイック作成

1. ダッシュボード上部の入力欄にタスク名を入力します。
    - 例: `日報を書く`
2. `Enter` キーを押します。
3. タスクがリストの先頭（現在のセクション）に追加されます。

## 詳細作成

1. 右上の `+` ボタンをクリックします。
2. モーダルが開くので、詳細を入力します。
    - **タイトル**: タスクの名前
    - **見積時間**: 予想される作業時間（分）
    - **プロジェクト**: 紐付けるプロジェクト
    - **タグ**: 分類用のタグ
3. `Save` ボタンをクリックします。
