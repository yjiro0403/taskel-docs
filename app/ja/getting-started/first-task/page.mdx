
# 最初のタスクを作る

タスクの作成は非常にシンプルです。

1. 右下の `+` ボタンをクリックします（またはショートカット `N`）。
2. モーダルが開くので、詳細を入力します。
    - **タイトル**: タスクの名前
    - **見積時間**: 予想される作業時間（分）
    - **プロジェクト**: 紐付けるプロジェクト
    - **タグ**: 分類用のタグ
3. `Save` ボタンをクリックします。

## AIでタスクを作る

AIチャットパネル（右下のSparklesボタン）から自然言語でタスクを作成することもできます。

- 「14時からMTG資料作成、30分で」→ AIがタスク候補を提案
- 内容を確認して「作成」ボタンを押すだけ

詳しくは [AIアシスタント](/ja/features/ai-assistant) をご覧ください。
