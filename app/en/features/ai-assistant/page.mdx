## Adding Tasks

Simply tell the AI what you want to do, and it will suggest a task as a preview card.

**Example: "Review the proposal at 2pm, about 45 minutes"**

The AI automatically determines:
- **Task name**: "Review the proposal"
- **Start time**: 14:00
- **Estimated time**: 45 minutes
- **Section**: Automatically placed in the Afternoon section based on the time

Review the suggestion card and press **"Create"** to add it. You can edit the title, time, or other details before creating.

![AI Task Suggestion Card](/images/ai-task-suggestion.png)

### Start Immediately Mode

Phrases like "**Start** email replies **now**" or "**Interruption** â€” phone call" will create the task and auto-start the timer. Any running task is automatically paused, making interruptions smooth.

## Goal Integration

### Viewing Goals

Ask **"What are my goals this month?"** to see all active goals â€” yearly, monthly, and weekly.

### Goal Breakdown

Tell the AI **"Break down this goal into tasks"** and it will:

1. Analyze the goal's content and check existing tasks
2. Suggest ~5 new tasks that don't duplicate existing ones
3. Show a batch preview for you to check and select which to create

Each task is automatically linked to the goal, so task completion is reflected in goal progress.

## Time Calibration

Ask **"Check my estimation accuracy"** to see a report analyzing gaps between your estimates and actual time spent.

### Calibration Report Includes

- **Accuracy ratio**: Actual / Estimated (100% is ideal)
- **Average deviation**: How far off your estimates are on average
- **Most deviated tasks**: Top 5 biggest gaps
- **Category accuracy**: Trends by tag

![Calibration Report](/images/ai-calibration.png)

### How It Helps

When calibration data is available, the AI automatically adjusts time estimates when suggesting tasks.

> Example: If you tend to exceed estimates by 50%, the AI might say:
> "This type of task usually takes about 45 minutes. Shall I estimate 45 minutes?"

## FAQ

### Will the AI create tasks without my permission?

No. The AI only makes "suggestions". A preview card is shown, and nothing is written to the database until you press the "Create" button. You can dismiss any suggestion with the "Reject" button.

### Which AI model is used?

Gemini 2.5 Flash is used by default. You can switch models using the selector in the chat panel.

### Is my conversation history saved?

Conversations are displayed during the session but reset when you close the panel. Only tasks you explicitly create are saved to Firestore.
