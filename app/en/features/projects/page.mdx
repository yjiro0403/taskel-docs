## Creating a Project

1. Click the `+` button in the **"Projects"** section of the sidebar.
2. Enter the project name.
    - You can also set a color and icon to improve visibility.
3. Add a description (Markdown supported) and milestones to organize the project's overall picture.

## Project Details

![Project Detail](/images/project-detail.png)

The project detail screen shows:

- **Milestones**: Manage major checkpoints within the project. You can set start date, end date, and status.
- **Task List**: Display progress of tasks linked to the project.
- **Members**: List of invited members.

## Inviting Members

In Taskel, you can invite members to collaborate on each project.

1. Open the project settings screen.
2. Select the **"Members"** tab.
3. Enter the email address of the person you want to invite and click `Invite`.
4. Alternatively, generate an invitation link with **"Copy Link"** and share it via chat tools.
