
# Project Management

Manage tasks by project and visualize progress.

## Creating a Project

1. Click the `+` button in the **"Projects"** section of the sidebar.
2. Enter the project name.
    - You can also set a color and icon to improve visibility.

## Inviting Members

In Taskel, you can invite members to collaborate on each project (Pro Plan, etc.).

1. Open the project settings screen.
2. Select the **"Members"** tab.
3. Enter the email address of the person you want to invite and click `Invite`.
4. Alternatively, generate an invitation link with **"Copy Link"** and share it via chat tools.
