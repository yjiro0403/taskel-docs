
# Notes & Review

This feature is for recording daily insights and reflections.

## Daily Notes

- Click the **document icon** on the date bar in Daily View to open the note for that day.
- You can write freely in Markdown format.
- It is saved automatically, so no save operation is required.

## Creating a Report

- Click the **"Copy Report"** button on the note screen to copy that day's "Completed Tasks" and "Note Content" as text to the clipboard.
- Useful for pasting into Slack, etc., as a daily or weekly report.

## Periodic Review

- **Weekly View**: You can create weekly reflection notes. Record this week's achievements and highlights.
- **Monthly View**: Monthly reflection notes. Reflect on themes and goal progress.
- **Yearly View**: Record annual aspirations and major reflections.
