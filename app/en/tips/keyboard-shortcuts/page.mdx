
# Keyboard Shortcuts

## Navigation

- `D`: Daily View
- `W`: Weekly View
- `M`: Monthly View
- `Y`: Yearly View
- `P`: Planning View
- `A`: Analytics (If implemented)

## Task Operations

- `N`: Create new task
- `Cmd/Ctrl + Enter`: Save task creation/edit
- `Esc`: Close modal

## Data Operations

- `Cmd/Ctrl + Z`: Undo
- `Cmd/Ctrl + S`: Manual Save (Usually auto-saved)
