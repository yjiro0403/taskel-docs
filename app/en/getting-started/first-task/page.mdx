
# Create First Task

Creating a task is very simple.

## Quick Creation

1. Enter the task name in the input field at the top of the dashboard.
    - Example: `Write daily report`
2. Press the `Enter` key.
3. The task will be added to the top of the list (current section).

## Detailed Creation

1. Click the `+` button on the top right.
2. A modal will open, so enter the details.
    - **Title**: Name of the task
    - **Estimate**: Estimated working time (minutes)
    - **Project**: Project to associate with
    - **Tags**: Tags for classification
3. Click the `Save` button.
